CSS & HTML
- Cơ bản:
  + cách dùng padding - margin
  + cách dùng đơn vị px, %, em, rem, fr, vh, vw, dvh
  + cách dùng 4 thuộc tính position: absolute, relative, sticky, fixed
  + cách dùng box-sizing: border-box và content-box
  + cách dùng border và outline
  + cách dùng các thuộc tính của display: block, inline, inline-block, flex, grid, none,... (không cần học table)
  + cách dùng flex, order, flex-wrap, flex-direction, self-align, gap trong flex
  + phân biệt display: none, visibility: hidden, opacity: 0
  + cách dùng một số thuộc tính: border-radius, box-shadow, z-index, background
  + cách dùng CSS selector cơ bản: div, .class, #id, [target="attr"],...
  + cách dùng các properties liên quan đến text như font-size, font-weight, font-family
  + cách dùng các properties liên quan đến list: https://www.w3schools.com/css/css_list.asp
  + cách dùng overflow: scroll, hidden và text-overflow: ellipsis
  + cách tính độ ưu tiên CSS dựa vào point, inline style và !important
  + style cho image (object-fit,...)
  + cách dùng cursor-pointer
- Nâng cao:
  + cách dùng @keyframe và animations
  + cách dùng transition kết hợp pseudo :hover  để tạo animation đơn giản
  + cách dùng cách thuộc tính transform: transition, scale, rotate và kết hợp @keyframe vs transform rotate để tạo ra loading animation
  + cách dùng CSS Combinators: https://www.w3schools.com/css/css_combinators.asp
  + cách dùng các pseudo như :after, :before, :focus, :hover,...
  + cách dùng @media để responsive
  + cách sử dụng variable trong CSS
  + cách sử dụng hàm calc()
  + cách custom lại style của scroll bar và placeholder của thẻ input


JS
- Cơ bản
  - Phân biệt const, let, var. Từ đó hiểu được function scope và block scope và hoisting
  - Cách dùng các methods của array: filter, map, forEach, some, every, reduce
  - Cách tương tác với DOM element thông qua addEventListener, querySelector, getElementById,...
  - Tìm hiểu về clousure, từ đó hiểu được cơ chế của Debounce và Throttle
  - Tìm hiểu về template string + innerHTML
  - Cách dùng Objecs.keys(obj), Objecs.values(obj), Objecs.entries(obj)
  - Tìm hiểu về local storage, session storage và cách dùng
  - Tìm hiểu cách truyền dữ liệu giữa các trang web trên cùng trình duyệt: localstorage, session storage, query pamrams,...
  - Tìm hiểu về querySelector và getElementById, từ đó thao tác với DOM thông qua add, remove class hoặc sửa inline style
  - Cách dùng Promise, Async Await, callback, so sánh ưu nhược điểm
  - Phân biệt shallow copy, deep copy
  - Cách sử dụng fetch để call API
- Nâng cao:
  + Biết cách triển khai tự call lại API sau khi call API fail với thời gian chờ để call lại tăng dần theo dãy số Fibonacci
  - Tìm hiểu về capturing vs bubbling khi add event vào DOM
  - Biết cách triển khai code tuân theo SOLID và DRY
  - Biết cách triển click outside event
